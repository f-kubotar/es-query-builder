    class EsQueryBuilder

    The class has a responsibility for converting a query string into a corresponding query hash object for Elasticsearch.

    Examples¶ ↑

    builder = EsQueryBuilder.new(
  query_fields: ['query'],
  filter_fields: ['filter']
)
# => #<EsQueryBuilder>

builder.build('term')
# => { match: { '_all' => 'term' } }

builder.build('query:term')
# => { match: { 'query' => 'hello' } }

builder.build('filter:term')
# => {
#      filtered: {
#        query: { match_all: {} },
#        filter: { term: { filter: 'hello' } }
#      }
#    }

builder.build('query\:term')
# => { match: { '_all' => 'query\:term' } }

builder.build('unknown:term')
# => { match: { '_all' => 'term' } }

    all_query_fields

    A String or an Array of Strings for searching usual query terms (default: '_all').

    hierarchy_fields

    An Array of Strings which treats the trailing slash character as a hierarchy (default: []).
